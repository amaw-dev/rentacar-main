# Plan: Red de Sitios Satélite para Linkbuilding

**Fecha:** 2026-01-18
**Estrategia:** Tiered Link Building con sitios propios
**Riesgo para sitio principal:** Muy bajo

---

## Concepto

```
TIER 3 (basura)          TIER 2 (satélites)       TIER 1 (principal)
─────────────────────────────────────────────────────────────────────
Links baratos            Sitios propios           alquilatucarro.com
de Cherada, Fiverr,  ──▶ que reciben los     ──▶ recibe links
PBNs económicos          links tóxicos            "limpios"

Si penalizan ────────▶   Satélite muere    ──▶   Principal intacto
                         (solo link saliente)
```

| ❌ NO HACER | ✅ HACER |
|-------------|----------|
| Mismo hosting para todos | Hostings diferentes (Hostinger, SiteGround, Namecheap) |
| Mismo registrante WHOIS | WHOIS privacy activado |
| Misma plantilla WordPress | Themes diferentes |

| Interconectar satélites | NUNCA enlazar entre satélites |
| Contenido AI puro | Contenido editado/humanizado |
| Crear todos el mismo día | Espaciar creación 2-4 semanas |

## Estrategia de Links

### Tier 3 → Tier 2 (Links baratos a satélites)

| Fuente | Costo | Cantidad | Frecuencia |
|--------|-------|----------|------------|
| Cherada guest posts | $20-50 | 5-10 | Mensual |
| Fiverr backlinks | $5-20 | 20-50 | Mensual |
| Comentarios en blogs | Gratis | 10-20 | Semanal |
| Perfiles web 2.0 | Gratis | 5-10 | Una vez |
| Directorios basura | Gratis | 20-30 | Una vez |

**Distribución por satélite:**
- No todos los links al mismo satélite
- Rotar entre los 5 sitios
- Variar anchor text (marca, genérico, URL)

### Tier 2 → Tier 1 (Satélites a principal)

| Tipo de link | Ejemplo | Frecuencia |
|--------------|---------|------------|
| Mención contextual | "empresas como alquilatucarro.com ofrecen..." | 1-2 por artículo |
| Recurso recomendado | "Para alquilar: alquilatucarro.com" | En guías |
| Comparativa | "Los mejores servicios incluyen..." | Ocasional |

**Reglas:**
- Máximo 1-2 links por artículo al principal
- Anchor text variado (70% marca, 20% URL, 10% keyword)
- También enlazar a otros sitios (Wikipedia, competidores, fuentes)

## Monitoreo y Métricas

### KPIs por Satélite

| Métrica | Objetivo 6 meses |
|---------|-----------------|
| Páginas indexadas | 20+ |
| DA (Moz) | 10-15 |
| Tráfico orgánico | 100+ visitas/mes |
| Links entrantes | 50+ |

### Señales de Alerta

| Señal | Acción |
|-------|--------|
| Desindexación | Pausar links, revisar contenido |
| Caída de DA | Reducir links tóxicos |
| Penalización manual | Abandonar satélite, crear nuevo |
| Links detectados como spam | Desautorizar en GSC del satélite |

---

## Contingencia: Si Penalizan un Satélite

1. **NO entrar en pánico** - el principal está protegido
2. Verificar que el link saliente a alquilatucarro no tenga atributos tóxicos
3. Opcionalmente: remover el link desde el satélite penalizado
4. Crear nuevo satélite para reemplazar
5. Continuar estrategia con los demás

## Notas Finales

- **Paciencia**: Resultados visibles en 3-6 meses
- **Consistencia**: Mejor poco constante que mucho de golpe
- **Naturalidad**: Los satélites deben parecer sitios reales
- **Diversificación**: No depender de un solo satélite
- **Documentación**: Registrar todos los links creados
